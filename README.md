# Aerial 

New repository for the upcoming new version of [Aerial screen saver](https://github.com/JohnCoates/Aerial/) for macOS compatible with macOS Tahoe. 

⚠️⚠️⚠️ Currently available as pre-release alpha versions, some features from the old screensaver are not working, there are bugs, etc... 

## Installation

1. Download the [latest release](https://github.com/AerialScreensaver/Aerial/releases) 
2. Unzip and move `Aerial.app` to `/Applications` (if you used to have a previous version of Aerial Companion, you can remove it)
3. Once copied, double click to launch `Aerial.app` in your Applications Folder 
4. If you previously used Aerial, you'll be prompted to migrate settings and data

## Uninstall

- To uninstall simply delete `Aerial.app` from your Applications folder. 
- To remove all data/settings, simply delete `/Users/Shared/Aerial`
